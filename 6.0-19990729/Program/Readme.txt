                        五子棋  版本 6.0
                        ================

    本游戏是一个免费软件，你可以复制和使用它而不必向作者支付任何费用或事先通知作者。
    本游戏使用的编程语言是Borland Pascal 7.0及其内嵌汇编，如果有错误报告、改进建议或者想要程序的源代码，可与作者联系。

                            一、简介
    你所得到的这个版本是本游戏的第六版。
    本游戏各版本简介如下：
        版本      完成日期      编程语言
        V1.0   1996年11月16日   Visual BASIC 3.0
        V2.0   1997年11月29日   Quick BASIC 4.5
        V3.0    1998年2月1日    Borland Pascal 7.0
        V4.0    1998年5月5日    Visual BASIC 3.0
        V5.0    1998年8月6日    Borland Pascal 7.0
    本版本是在第五版的基础上完成的。本版本较前一版本改进了游戏界面、支持使用鼠标对弈和禁手并支持棋谱的保存和打印，在人工智能方面也有所改进。
    本游戏由以下五个文件组成：
        GOBANG.EXE  程序文件
        GOBANG.DAT  数据文件
        README.TXT  说明文件
        MANUAL.TXT  棋谱文件
        HISTORY.DAT 历史记录
    这其中后两个文件是程序执行时生成的。

                          二、使用方法
    执行GOBANG.EXE即可启动本游戏。
    要执行本游戏，你应该保证你的系统有：
        80286或更好的处理器，建议使用486DX/66或更好的处理器；
        VGA彩显，至少256KB显存；
        鼠标器(可选)；
        打印机(可选)。
    此外，你必须保证有至少560KB的连续剩余常规内存片和几十KB的剩余磁盘空间。
    如果在软盘上执行，最好加载SMARTDRV程序。
    在游戏的主菜单中，有六个项目：
    ※人机对弈
        选择此项后，游戏将首先询问姓名。若要输入中文，可以按：
          快捷键    输入法
        <ALT>+<F1>   区位
        <ALT>+<F2>   拼音
        <ALT>+<F6>   英文
        姓名的长度应在1到24个字符之间，不能使用“计算机”作为姓名。
        输入了姓名后，将选择由谁先行和要使用的输入设备。选择好后就可以开始游戏了。
        游戏支持鼠标和键盘两种输入设备。对于键盘，又有方向键和字母键两种输入方式：
          功能  方向键  字母键
           上   【↑】  【Ｗ】
           下   【↓】  【Ｓ】
           左   【←】  【Ａ】
           右   【→】  【Ｄ】
          确认 【回车】【空格】
        比赛结束后，游戏可以将本次比赛的棋谱保存到文件MANUAL.TXT中或打印到打印机上。
    ※双人对弈
        请参见“人机对弈”部分的说明。
    ※历史记录
        游戏会保存所有进行过的比赛的结果，可以用此项来浏览它们。
        删除游戏所在文件夹的HISTORY.DAT文件即可清除历史记录。
    ※使用说明
        选择此项可以阅读使用说明。
    ※版权信息
        选择此项可以查看版权信息。
    ※退出游戏
        选择此项可以返回操作系统。

                          三、游戏规则
    五子棋，亦称“串珠”、“连五子”，日文称之为“五石”、“ごもくなら”、“れんじゅ”，英文则称之为“Gobang”、“mo-rphion”、“Renju”或“FIR(Five In A Row的缩写)”，是中国民间非常熟知的一个古老棋种。相传，它起源于四千多年前的尧帝时期，比围棋的历史还要悠久。
    五子棋这一活动在其发展中经过不断的改良，其重要的规则改革包括：
        1899年规定，禁止黑白双方走“双三”
        1903年规定，只禁止持先的黑方走“双三”
        1912年规定，如果黑方被逼迫走成“双三”，那么，黑方就立刻输了
        1916年规定，禁止黑方走长连
        1918年规定，禁止黑方走“四、三、三”，但是，黑方若走成“五、三、三”则为赢棋
        1931年规定，禁止黑方走“双四”，同时提议把五子棋的棋盘从19×19的围棋棋盘改为15×15的五子棋专用棋盘
        ……
    五子棋棋盘为十五路，呈正方形，平面上画横竖各15条平行线，线路为黑色，构成225个交叉点，棋盘正中一点为“天元”，周围四点为“小星”。
    五子棋比赛规则如下：
    ※黑先、白后，从天元开始相互顺序落子
    ※最先在棋盘横向、竖向、斜向形成连续的相同色五个棋子的一方为胜
    ※黑棋禁手判负，白棋无禁手。黑棋禁手包括“三、三”、“四、四”。“长连”判负
    ※如分不出胜负，则定为平局
    ※对局中拔子、中途退场均判为负
    ※五连与禁手同时形成，先五为胜
    ※黑方禁手形成时，白方应立即指出。若白方发现而继续应子，不能判黑方负

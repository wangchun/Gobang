                               五子棋游戏使用说明
    你所得到的这个版本是本游戏的第五版。
    本游戏各版本简介如下：
  ┏━━━┯━━━━━━━┯━━━━━━━━━┯━━━━━━━━━━━━━━┓
  ┃版  本│   完成日期   │     编程语言     │       重  要  改  进       ┃
  ┠───┼───────┼─────────┼──────────────┨
  ┃第一版│1996年11月16日│ Visual BASIC 3.0 │Windows版本，本游戏最初版本 ┃  
  ┠───┼───────┼─────────┼──────────────┨
  ┃第二版│1997年11月29日│ Quick BASIC 4.5  │DOS版本，改为标准15X15的棋盘┃
  ┠───┼───────┼─────────┼──────────────┨
  ┃第三版│ 1998年2月1日 │Borland PASCAL 7.0│DOS版本，增加了人机对弈功能 ┃
  ┠───┼───────┼─────────┼──────────────┨
  ┃第四版│ 1998年5月5日 │ Visual BASIC 3.0 │Windows版本，支持禁手和长连 ┃
  ┗━━━┷━━━━━━━┷━━━━━━━━━┷━━━━━━━━━━━━━━┛
    本版综合了第三版和第四版的优点，对算法进行了很大改进，首次使用了博弈树搜索技术，使用了扩充内存(XMS)，去掉了鼠标支持和并不怎么受欢迎的禁手，从而降低了程序的复杂度，提高了速度。
    本游戏对系统的要求不高，只需要一台VGA显示的286微机和2M内存即可执行，但如果要与“困难”级别的计算机对弈，最好有一块奔腾级的CPU。
    整个游戏由下表所示的几个文件组成：

    除README.TXT外，缺少任何一个文件都会使游戏不能正常执行，所以在执行游戏前请仔细检查游戏文件是否完整。
    将上述文件复制到同一目录中，并在该目录下键入“WZQ”即可启动游戏。在主菜单中选择相应的项并按回车键，然后按游戏的提示做即可。当游戏提示输入姓名时，可按<ALT>+<F1>键启动区位码输入法、按<ALT>+<F2>键启动拼音输入法或按<ALT>+<F6>键退出中文输入。
    如果是人机对弈，那么下棋时应使用光标键和回车键；如果是双人对弈，黑方则改用A、D、W、S四键以代替光标键，用空格代替回车键。游戏中可随时按ESC键认输以退出游戏。
    如果你发现游戏中存在错误，或者对游戏有好的改进意见，请通知本游戏的作者，以使本游戏得以进一步的改进。
